# Anti-AI Voice Auditor

Catch AI writing patterns before your readers do.

## What's Inside

**3 skills, 1 plugin.**

### AI Writing Audit
Scans content for 16+ categories of AI tells: clichés, inflated phrases, weasel words, false balance structures, empty modifiers, marketing fluff, and more. Outputs flagged issues with exact quotes, category tags, and corrected rewrites.

### LinkedIn Hook Writer
Generates 10 pattern-breaking hooks using 5 techniques: contradiction, specific numbers, direct accusation, stolen thought, and absurd reframe. Character-counted for LinkedIn's "... more" cutoff.

### Voice Guide Template
A structured template to define your authentic writing voice — tone, sentence rhythm, platform rules, red-flag phrases, and a 3-question decision test. Works as the drafting layer before the audit catches what slips through.

## How They Work Together

```
Draft (voice guide) → Format (platform skill) → Audit (pattern detection) → Hooks (LinkedIn)
```

Each skill works standalone, but the pipeline is where the magic is.

## Install

```bash
# Add to your repo's marketplace
mkdir -p .agents/plugins
# Then add plugin to your marketplace.json (see Codex docs)
```

## Origin

Built from a production content pipeline that runs daily. The audit categories come from analyzing 100+ AI-generated drafts. The hook techniques come from Ruben Hassid's framework. The voice guide template comes from 6 months of iterating on authentic writing with AI assistance.
